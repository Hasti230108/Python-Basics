# Day 08 - Object-Oriented Programming (OOP)

## Topics Covered

- Class
- Object
- Constructor (`__init__()`)
- Methods
- Inheritance
- Encapsulation

## Files

### class_object.py
- Created a class
- Created objects
- Accessed object attributes

### constructor.py
- Learned how constructors initialize object data
- Used the `self` keyword

### methods.py
- Created class methods
- Accessed object data using methods

### inheritance.py
- Created a parent class (`Person`)
- Created a child class (`Student`)
- Used `super()` to call the parent constructor
- Inherited methods from the parent class

### encapsulation.py
- Created private attributes using `__`
- Accessed private data using a getter method

## Concepts Learned

- Difference between Class and Object
- Constructor (`__init__`)
- `self` keyword
- `super()` keyword
- Parent and Child classes
- Code Reusability through Inheritance
- Data Hiding using Encapsulation

## Outcome

Today I learned the fundamentals of Object-Oriented Programming (OOP) in Python. I practiced creating classes and objects, using constructors and methods, implementing inheritance with `super()`, and protecting data using encapsulation. These concepts form the foundation for building scalable and reusable Python applications.
